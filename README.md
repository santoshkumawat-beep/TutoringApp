# TutoringApp

## Project Documentation

### Introduction
TutoringApp is an application designed to connect students with tutors for various subjects. The app allows students to find tutors based on their specific needs and schedule sessions accordingly.

### Features
- **User Registration**: Students and tutors can create accounts.
- **Search Functionality**: Students can find tutors by subject, rating, and availability.
- **Session Scheduling**: Users can book sessions directly through the app.
- **Feedback System**: Students can leave reviews for tutors after sessions.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/santoshkumawat-beep/TutoringApp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TutoringApp
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Usage
To start the application, run:
```bash
npm start
```
Visit `http://localhost:3000` in your browser to access the app.
